# 🚀 Productivity Dashboard

A modern, responsive, and minimal productivity dashboard built using **HTML, CSS, and JavaScript**. Designed to help users stay organized, focused, and productive with a clean grayscale UI and smooth interactions.

## 🌐 Live Demo

👉 https://as3148300-prog.github.io/Productivity-Dashboard/

---

## ✨ Features

* 🎯 Clean & Modern UI
* 🌙 Dark / Light Theme Toggle
* 📝 Task Management
* ⏱️ Pomodoro Timer
* 📅 Calendar
* ⏰ Digital Clock
* 📊 Productivity Dashboard
* 💾 Local Storage Support
* 📱 Fully Responsive Design
* ⚡ Smooth Animations
* 🎨 Minimal Grayscale Theme

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/AS3148300-Prog/Productivity-Dashboard.git
```

Open the project

```bash
cd Productivity-Dashboard
```

Run the project by opening `index.html` in your browser.
